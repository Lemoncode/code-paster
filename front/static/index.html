Code paster is a free online app

What can I use it for?

You can use Code Paster to share code or any kind of text with partners, students, colleagues, etc. The trainer role (who can be one or more people) is allowed to write and read text. However, the student role is only allowed to read it. This particularity makes Code Paster a great and simple application to be used in any kind of online activity which requires text to be shared.

How does it work?

  1. Press "Get started" button to navigate to the app
  2. Press "Create session" button to start a private room for you and your people
  3. Share the links on the top of the website which other people, taking care of use "student" or "trainer" role as needed
  4. Start typing within the "Trainer Textarea" and the other people in the room will be able to see whatever you have writen in it
